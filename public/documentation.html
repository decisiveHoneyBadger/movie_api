      ]

Allow users to update their user name /users/[username] PUT :Username A JSON object holding the new user name. Ex: 
              
    { 
      "name": "Kim"
    }
JSON object with the updated user. Ex: 
              
    { 
      id: '1',
      name: 'Kim'
    }
Allow users to add a movie to their list of favorites /users/[username]/movies/[movieid] POST :Username, :MovieID None Movie ID added to user's list of favorites
Allow users to remove a movie from their list of favorites /users/[username]/movies/[movieid] DELETE :Username, :MovieID None Movie ID gets removed from user's list of favorites
Deletes a user /users/[username] DELETE :Username None Message stating user has been deleted
